# Memory 3D (Static / GitHub Pages)

## تشغيل على GitHub Pages
1) ارفع الملفات كما هي (index.html + main.js + textures.js + style.css).
2) من Settings → Pages:
   - Source: Deploy from a branch
   - Branch: main
   - Folder: / (root)
3) افتح رابط GitHub Pages.

## تشغيل محلي (اختياري)
افتح index.html مباشرة في المتصفح.
(إذا منع المتصفح ES Modules محليًا، شغّله عبر Pages أو عبر أي Static Server.)
